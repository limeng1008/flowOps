## 五、工程铁律(踩坑沉淀)

1. **Node 20 强制**(`nvm use 20`,系统默认 22 会炸 native 模块)
2. **只做加法不改核心**:新能力 = 新节点/独立服务/新页面;接入点最小化(可持续 merge 上游)
3. **新实体日期列必须 `type: Date`**,禁止 `'datetime'`/`'timestamp'` 字符串(守护测试拦截)
4. **新实体必写 4 库 migration**(postgres/mysql/mariadb/sqlite),时间戳避撞
5. **加依赖禁用 `pnpm add`**(连带升 @types/node 炸上游)——手改 package.json + `pnpm install`
6. **新节点必须 build dist + 重启后端**才会出现在画布(踩过多次)
7. **模板表单变量必须 `{{ $form.字段 }}`**;模板结构测试绿 ≠ 能跑,新模板须真机验证
8. **钱按分存整数;密钥只读 env;回调验签失败即拒;订单状态机只进不退**
9. **i18n en+zh 同步加;品牌官方名不翻译;nodeI18n 防重复键**
10. **多智能体并发**:Codex 与 Claude 共用工作区,重要改动**立即 commit**(未提交的被重置过)
